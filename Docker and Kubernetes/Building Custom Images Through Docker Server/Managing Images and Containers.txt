 - On any Docker command, you can use --help to see all options
 - Images
    - Can be tagged
        - -t, docker tag
    - Can be listed
        - docker images
    - Can be analyzed
        - docker image inspect
    - Can be removed
        - docker rmi, docker prune
 - Containers
    - Can be named
        - --name
    - Can be configured in detail
        - see --help
    - Can be listed
        - docker ps
    - Can be removed
        - docker rm
